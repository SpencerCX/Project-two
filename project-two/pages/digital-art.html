Shown on this page are four of the projects that I completed in Digital Drawing class and one project that I completed in Color Theory class this semester.



Project 1

This focused on drawing a pair of headphones. In this project, I learned about using shadows and layers.

Project 2

This focused on drawing a man dancing. In this project, I learned about drawing poses and stances.

Project 3

This focused on drawing six different items with different textures. I drew a curtain, golf ball, rocks, glass with water, a tree with a heart, and a spoon. I learned about how to draw different textures.

Project 4

This focused on drawing toaster from four different views. I drew the toaster from the front, back, side, and top. I learned about different perspectives.

Project 5

This focused on creating a digital version of the handmade color whee. I learned how to do this using photoshop.
